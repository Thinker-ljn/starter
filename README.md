# Starter

## Rename `starter` to your package name

- .github/workflows/release-tag.yml
- package.json

## Write your code

- src/\_\_tests\_\_
- index.ts

## Try it now

### Use GitHub Template

[Use GitHub Template](https://github.com/Thinker-ljn/starter/generate).

### Clone to local

```bash
npx degit Thinker-ljn/starter my-starter-app
cd my-starter-app
# npm install -g pnpm
pnpm i
```
